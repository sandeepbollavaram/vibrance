What's inside

Batch editing

Apply your edit pipeline to a whole folder. Runs off the UI thread — drag a hundred files in, keep working.

Smart compression

Target a file size in KB. Vibrance binary-searches encoder quality until your image lands at or under the limit. Strips EXIF on request.

Before / after compare

Draggable split view with labeled handles. See exactly what changed.

11 adjustments

Exposure · Brightness · Contrast · Highlights · Shadows · Saturation · Vibrance · Temperature · Tint · Blur · Sharpen — with reset on every slider.

Presets & LUTs

Save any look as a preset. Drop in a .cube 3D LUT and apply it across a folder.

Built for keyboards

Undo/redo, Apply, Fit, Compare, Export — all shortcuts. Live histogram updates as you drag.

From source

macOS / Linux or hacking on the code? Run from source.

git clone https://github.com/sandeepbollavaram/vibrance.git
cd vibrance
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m image_editor
